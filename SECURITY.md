# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 5.0.x   | :white_check_mark: |
| < 5.0   | :x:                |

## Reporting a Vulnerability

We greatly appreciate the community that supported our project to keep it secure!

Security and privacy are our top commitments for the project. If any issues are discovered, please use our chat support system on our website to be connected directly with our development team. You may also contact us in our Discord server.

[OctoEverywhere Support](https://octoeverywhere.com/support?source=github_security_md)
[OctoEverywhere Discord](https://octoeverywhere.com/r/discord?source=github_security_md)

Please provide as much information as possible in your report.
